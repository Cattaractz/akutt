# ABCDE: Primærundersøkelse
Undersøkelsene etter hvordan de dukker opp i HP


Denne guiden følger standard ABCDE-metodikk for akuttvurdering av barn.

---
<div class="desktop-view">

### **A** – Luftveier (Airway)
* **Status:** Frie, Truet eller Ufrie?
* **Tiltak:** Sørg for frie luftveier ved behov.

---

### **B** – Ventilasjon (Breathing)
| Parameter | Observasjon |
| :--- | :--- |
| **RF** | Respirasjonsfrekvens |
| **SpO2** | Metning (mål: >94%) |
| **O2-Behov** | Bruker barnet romluft eller O2? |
| **Utstyr** | Hvilket utstyr brukes til å gi O2? |

| Respirasjonsmønster og lyder |
* **Mønster:** Uanstrengt, takypne, uregelmessig eller pressende/stønnende.
* **Ekspirie:** Er det forlenget ekspirie? 
* **Inndragninger:** SC, IC, jugulære, nesevingespill eller headbobbing.
* **Muskulatur:** Bruk av hjelpemuskulatur.
* **Lyder:** Respirasjonslyder (stridor, pipelyd, knatrelyd).

#### **Cyanose**
* [ ] Ingen
* [ ] Perifert (hender/føtter)
* [ ] Sentralt (lepper/munnhule)

---

### **C** – Sirkulasjon (Circulation)
* **Vitale mål:** Blodtrykk, Puls, Temp, Kapillærfylling
* **Hudfarge:** Normal, blek, gusten, rødme, gul, eller marmorert?
* **Hudstatus:** Varm/kald, tørr/klam/svett

#### **Hydrering og Væskebalanse**
* **Hudturgor:** Vurder hudturgor
* **Øyne:** Normale eller innsunkne?
* **Slimhinner:** Fuktige, tørre, eller gråter barnet uten tårer?
* **Fontanelle:** Normal, spent, bulende, innsunken eller suturdiastase
* **Urinproduksjon:**  Normal, økt eller mindre enn vanlig 

---

### **D** – Nevrologi (Disability)
* **Bevissthet:** Vurderes via **AVPU**
| Grad | Forklaring |
| :--- | :--- |
| **A** (Alert) | Barnet er våkent og responsivt. |
| **V** (Voice) | Barnet reagerer på tiltale/stemme. |
| **P** (Pain) | Barnet reagerer kun på smertestimuli. |
| **U** (Ureaktivt) | Barnet reagerer ikke. |

* **Skåring:** Bruk GCS (Glasgow Coma Scale) ved behov

---

### **E** – Eksponering / Annet
* Sjekk hele kroppen for utslett, traumer eller andre funn

<div class="mobile-view">

### **A** – Luftveier (Airway)
* **Status:** Frie, Truet eller Ufrie?
* **Tiltak:** Posisjonering, suging, svelgtube/maske.

---

### **B** – Ventilasjon (Breathing)
* **Status:** RF, SpO2, O2-behov.
* **Mønster:** Inndragninger, bruk av hjelpemuskulatur.
* **Lyder:** Stridor, hvesing, knatrelyder.
* **Cyanose:** Ingen, perifer eller sentral?

---

### **C** – Sirkulasjon (Circulation)
* **Vitalia:** Puls, BT, Temp, Kapillærfylling.
* **Hud:** Normal, blek, gusten, gul, marmorert, varm/kald, tørr/svett.
* **Hudstatus:** Varm/kald, tørr/klam/svett

#### **Hydrering og Væskebalanse**
* **Hudturgor:** Vurder hudturgor
* **Øyne:** Normale eller innsunkne?
* **Slimhinner:** Fuktige, tørre, eller gråter barnet uten tårer?
* **Fontanelle:** Normal, spent, bulende, innsunken eller suturdiastase
* **Urinproduksjon:**  Normal, økt eller mindre enn vanlig 

---

### **D** – Nevrologi (Disability)
* **Bevissthet:** AVPU (Alert, Voice, Pain, Unresponsive).
* **Pupiller:** Symmetri og lysreaksjon.
* **Blodsukker:** Mål alltid ved påvirket bevissthet!

---

### **E** – Eksponering (Exposure)
* **Hud:** Se etter utslett (petekkier!), traumer eller merker.
* **Temp:** Beskytt barnet mot nedkjøling under undersøkelsen.
